**Tugas Praktikum – Library App**
Buatlah aplikasi (Library App) sederhana berbasis Android dengan menerapkan
ketentuan-ketentuan dibawah ini:
**Fitur Aplikasi**

1. **Navigasi Bawah (Bottom Navigation)** terdiri dari:
    ○ **Home** : Menampilkan daftar semua buku.
    ○ **Favorites** : Menampilkan buku yang diberi Like.
    ○ **Add Book** : Form tambah buku baru.
2. **Data Buku** terdiri dari:
    ○ Judul, Penulis, Tahun Terbit, Blurb, Gambar (cover), dan status Like.
    ○ Dummy buku minimal 15.
3. **Halaman Home** :
    ○ Gunakan RecyclerView untuk menampilkan daftar buku.
    ○ Tambahkan SearchView untuk pencarian berdasarkan judul.
    ○ Klik buku → buka DetailActivity (menampilkan info lengkap + tombol Like).
    ○ Buku terbaru tampil di urutan atas.
4. **Halaman Favorites** :
    ○ Tampilkan daftar buku yang diberi Like.
    ○ Perbarui daftar setiap kali fragment ditampilkan.
5. **Halaman Add Book** :
    ○ Form input data buku.
    ○ Gambar cover diambil dari galeri.
**Ketentuan Teknis**
● Menerapkan tampilan UI yang menarik
● Tidak perlu menggunakan **database** atau **SharedPreferences**.
● Tambahan nilai jika menambahkan lebih banyak data di detail buku seperti rating,
genre, review, dll.
● Tambahan nilai jika membuat filter buku berdasarkan genre.
